# Video Style Presets

Each file defines a reusable video generation style.

## File Format

```markdown
# [Style Name]

## Base Prompt
[Core cinematic style: camera movement, color grading, film grain, frame rate feel.]

## Negative Prompt
[What to avoid: flickering, morphing, artifacts.]

## Recommended Tools
[Runway Gen-3, Sora, Kling, Veo, etc.]

## Parameters
[Duration, resolution, motion intensity, camera settings.]
```
