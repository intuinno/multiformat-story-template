# Character Motion Prompts

Each file defines how a character moves and acts for video generation.
Used with video style presets to create consistent character motion.

## File Format

```markdown
# [Character Name]

## Movement Style
[How they walk, run, fight, gesture. Prompt-ready motion description.]

## Acting Notes
[Emotional range, physical mannerisms, signature gestures.]

## Reference
[Links to corresponding image prompt: prompts/image/characters/[name].md]
```
