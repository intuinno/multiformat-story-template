# Voice Acting Prompts

Each file defines a character's voice for AI voice generation.

## File Format

```markdown
# [Character Name]

## Voice Description
[Pitch, timbre, accent, speech speed, emotional range.]

## Sample Lines
[Key dialogue lines with acting direction for calibrating the voice.]

## Recommended Tools
[ElevenLabs, etc.]

## Reference
[Links to corresponding character sheet: bible/characters.md]
```
