# Music Prompts

Each file defines a music piece or theme for audio generation.

## File Format

```markdown
# [Track Name]

## Purpose
[Where this plays: trailer, scene underscore, character theme, episode intro, etc.]

## Prompt
[Genre, mood, instruments, tempo, dynamics. Prompt-ready description.]

## Duration
[Target length.]

## Recommended Tools
[Suno, Udio, etc.]

## Reference Tracks
[Existing songs/scores that capture the intended feel.]
```
