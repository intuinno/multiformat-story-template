# Setting Visual Prompts

Each file defines a location's visual identity for image generation.
Compose with character + style fragments for complete scene prompts.

## File Format

```markdown
# [Location Name]

## Core Description
[The essential visual identity of this place. Prompt-ready.]

## Time Variations
[How it looks at different times: dawn, day, dusk, night, storm, etc.]

## Key Details
[Signature visual elements that make this location recognizable.]

## Camera Angles
[Suggested compositions: establishing wide, interior medium, detail close-up, etc.]
```
