# Reference Overview

모든 레퍼런스를 한눈에 훑어보기 위한 페이지.
Obsidian에서 열면 이미지가 인라인으로 렌더링된다.
레퍼런스가 추가될 때마다 이 파일도 함께 업데이트한다.

---

## 작성 가이드

접두사별로 섹션을 나누고, 캐릭터/장소별로 서브섹션을 만든다.
이미지는 3열 테이블로 배치한다.

```markdown
## [접두사] — [카테고리]

| | | |
|---|---|---|
| ![](images/파일명1) | ![](images/파일명2) | ![](images/파일명3) |
| NNN — 설명 | NNN — 설명 | NNN — 설명 |
```

빈 셀은 비워둔다.
레퍼런스가 아직 없으면 이 템플릿 내용을 지우고 실제 이미지로 채운다.
