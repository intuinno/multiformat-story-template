# References

레퍼런스 이미지와 비디오를 이 폴더에 넣는다.
파일명으로 소팅하면 용도별로 묶이고, 번호순으로 정렬된다.

## Naming Convention

```
[접두사]-[NNN]-[설명].[ext]
```

접두사:

| 접두사 | 용도 |
|--------|------|
| `char` | 캐릭터 비주얼 |
| `mood` | 분위기/톤 |
| `motion` | 모션/연출/카메라워크 |
| `setting` | 장소/배경 |
| `style` | 아트 스타일/채색 |

예시:
```
char-001-주인공-얼굴참고.jpg
char-002-주인공-전신.png
char-003-조연-체형참고.jpg
mood-001-야경-골목길.png
mood-002-새벽-안개.jpg
motion-001-액션-칼싸움.mp4
setting-001-폐건물-외관.jpg
setting-002-폐건물-내부.jpg
style-001-manhwa-채색참고.webp
style-002-cinematic-색보정.jpg
```
